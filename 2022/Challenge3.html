Lost in Transpiling

Now you’re dressed, it’s time for your interview. As you approach the building you realize you can’t read the building numbers or signs. It is in another language! To get to the correct room and building you must Transpile the foreign language.

Transpiling is the process of turning one software language into another. One application is ease of use. For example, TypeScript streamlines and improves writing JavaScript by making it type safe, and transpiles into JavaScript to maintain web compatibility. Another application is modernizing legacy software by transpiling an old, difficult to maintain language into something modern. A fake language is described in Documentation.txt, and a program is supplied in Program.txt. Write a program to transpile the supplied program into a language of your choice, write it to a file, and demonstrate that it can be run without any manual corrections.

Intel

Your transpiled language will have four variables with values. Those values should help you find the lot, building, floor and room.
