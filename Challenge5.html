Image Steganography

You have been provided with a raster image, “secret.png”. This image appears to be a beautiful vista of the Colorado School of Mines campus, but as the filename implies, there is a secret hidden within.

Your objective is to expose a covert drawing using inside details of the cipher used to hide it. The drawing was hidden using steganography: the practice of concealing a message within another message. In this case, the ordinary image is not only storing the key, but also the encrypted message you are tasked with decoding.

Every pixel of the visible image and its [X,Y] position exactly corresponds to a [X,Y] pixel in a hidden image you will construct. As such, the invisible image’s pixels are concealed within each pixel of the RGB-encoded picture. secret.png, the campus photo, is a 24-bit image comprised of red, green, and blue components. The encoding of the binary image color information can be thought to be stored like:

RRRRRRRR GGGGGGGG BBBBBBBB

However, the secret image is a grayscale/non-color image whose pixels are each constructed of a 3-bit color component. As we understand with binary numbers, keep in mind that this means the maximum value would be 2^3-1, therefore the maximum luminosity expressed by our hidden image will be 7. Let’s call the bits that make up this 3-bit number X, Y, and Z;

X (representing 4) is most significant, and Z (representing 1) is least significant.

These grayscale values and their corresponding luminosity bits (X, Y, and Z) have been respectively hidden within the least significant bit of the red, green, and blue color components in the visible image (secret.png). X has been hidden inside red, Y has been hidden inside green, and Z has been hidden inside blue. Remember, each pixel corresponds to a pixel in the hidden image, so each non-secret pixel contains all the information you need for the secret pixel in the exact same location.

RRRRRRRX GGGGGGGY BBBBBBBZ

There is additional protection, though. This secret image has been protected with an XOR cipher. To fully decrypt the image, you need to XOR the grayscale bits (X, Y, and Z) against the bit to the left of each one. So, for the pixel if the RGB value is:

00000011 00000001 00000010

Then you must perform and XOR operation, X = XOR (1, 1), Y = XOR (0, 1), and Z = XOR (1, 0)

Once you have the secret image, tell us what it is!
