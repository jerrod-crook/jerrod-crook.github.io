Disjointed Graphs

The attached file (points.txt) contains several points in 3D space, each on their own line. Within the total set of points, there are two disjoint graphs (i.e. the set of points A,B,C are connected, the set of points D,E,F are connected, but nothing connects these two sets directly).

Your objective is to find all of the valid points that will connect the disjoint graphs. There is no order to the master list of points, you will have to determine what the disjoint sets are.
