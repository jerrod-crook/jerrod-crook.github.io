Compromised User Accounts

You work at the industrious Super Secure Software Sellers TM Company.

Unfortunately, there has been a recent account breach from various sites across the web and a combination of usernames and passwords have been leaked out to the world.

You have been provided the plain csv file of usernames and passwords combinations that was posted to the web.

It is up to you to take a json file retrieved for you by a local IT administrator that contains pairs of usernames and encrypted passwords and match them to these compromised records.

You are made aware that passwords are encrypted via SHA256 hash and then converted into a base 64 string to store into the company’s system.


Part A
  • 1. Please provide a printout of all the users who are compromised. (Matching username and password)

Part B

  • 2. While we are going through this effort IT would like to know who within the system have the password of “password” or “Password” or “Password1234”?

    3. IT has also noticed that many people are using a very common password generator to protect their accounts. What users have passwords contained in the compromised file, but don't match on username?
