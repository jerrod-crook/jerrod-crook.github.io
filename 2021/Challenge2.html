Word Search

You've been hired by the New York Times to help improve their daily crossword puzzle. Your first task on the job is to build a puzzle verifier.

You will need to write a program that can read the puzzle file and the key file.

Not all keys are in the puzzle. You will need to correctly identify which keys are present in the puzzle and which are missing.

You will also need to determine the orientation of the keys. (Horizontal, Vertical, Diagonal Up, Diagonal Down)

  • Puzzle File
  • Key File
