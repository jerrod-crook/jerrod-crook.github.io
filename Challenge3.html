Password Manager

In recent news, it's been leaked that the NotSoFast Password Manager, instead of using modern encryption techniques, has opted to use a chain of basic cyphers for all of their passwords.

The pattern goes like this:

Given some integer n, perform these operations n times:

rotate the text n times to the left

shift letters up by n (i.e. a Caesar cipher)

For example, for n = 2:

Original Rotate left Shift up Rotate left Shift up
Password sswordPa uuyqtfRc yqtfRcuu AsvhTeww

Seeing the news, you decide to snag some cyphertext passwords off the dark web in exchange for some QubitCoin,

hoping to steal some personal info (even though your mother raised you better than that).

What are the plaintext versions of the passwords, and what value of n was used?

Note

1. the alphabet we're using is:

abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ1234567890

2. rotations at the end of a sequence wrap to the start.
